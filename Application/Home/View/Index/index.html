课程简介
COURSE INTRODUCEION
VR（Unity开发方向）
Unity 3D是VR应用内容开发的主流引擎，可以轻松如三维视频游戏、建筑可视化、实时三维动画等类型互动内容的多平台的综合型游戏开发工具，是一个全面整合的专业游戏引擎。
VR（UI设计方向）
UI设计主要涵盖平面、网页、游戏、Web前端、产品交互等多种设计，而VR的UI设计与传统的UI设计不同之处主要体现在交互设计上。
VR（室内设计方向）
室内设计以"创造满足人们物质和精神生活需要的室内环境"作为室内设计为目的。VR的室内设计课程，在传统的室内设计基础上，升级加入虚拟现实技术及交互设计课程。
VR（web前端方向）
Web前端开发，利用(X)HTML/CSS/JavaScript/Flash等各种技术进行客户端产品的开发，致力于通过技术改善用户体验。虚拟现实的实现离不开Web前端开发。
VR（游戏模型方向）
游戏模型应用于游戏场景制作、游戏角色制作、动漫影视模型制作。而虚拟现实应用有着高质量的画面要求，这使游戏模型行业发展迅猛、人才紧缺。
VR（建筑展示方向）
利用虚拟现实技术对现实中的建筑进行三维仿真，具有人机交互性、真实建筑空间感、大面积三维地形仿真等特性。能够获得身临其境的体验。

校园环境

华录文化教育学院，注重学生综合能力的培训，以虚拟现实培训为主体，衍生出与之相关的虚拟现实技术展示，VR项目实训，企业内训等，这里拥有北京、全国乃至世界的互联网领先技术。学生在这里不止可以学习到符合企业需求的专业技能，更可以收获无限可能。

住宿条件

我们为学生提供良好的住宿服务，宿舍内干净整洁，设施齐全，每个房间配有单独卫浴。房型设置有2人间、4人间、6人间、8人间，学生可以自行选择房型。如遇家人同行，这里还配有宾馆式标间方便住宿。

周边环境

中国华录文化教育学院建于华录集团大厦内，坐落于石景山区，石景山区是中国北京市西部的一个市辖区，东临帝阙，西濒浑河，距天安门16千米，因“燕都第一仙山石景山”而得名，是重工业企业、高科技单位和多个国家体育运动中心聚集的地方，有首都钢铁总公司等国有大中型企业数家，还拥有从幼儿园、小学、中学到大学的完善教育体系和完备的教育设施，荟萃天泰山、翠微山、石景山、莲花河等景点，是北京市城区中山林资源最丰富、绿化覆盖率最高、人均拥有公共绿地最多的地区。
